Get started today!

Coding can be for anyone and can enable you to enhance your career and day-to-day activities.
Below are some helpful sites and tips to kickstart your rendezvous with coding.



Free Code Camp


Complete numerous coding challenges at your own pace and connect with other coders nearby.



Codeacademy


This site contains tutorials on a variety of languages using exercises that help you gain first-hand experience.



Codewars


Learn how to code in a "dojo" environment and earn ranks/honors as you go.

Tips

  • Learn by doing - PRACTICE, PRACTICE, PRACTICE

  • Take a break and revisit code later when you run into a problem

  • Be patient - it won't be a "Eureka!" moment overnight

  • Ask for help when needed - the web, peer, or experts

  • Don't be afraid of making mistakes - you'll learn a lot from these

  • Continuously dive into new topics and languages
